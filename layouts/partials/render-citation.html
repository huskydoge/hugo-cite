{{/* Partial template for rendering a citation */}} {{- $key := .key }} {{- $keyIndex := .keyIndex }} {{- $reference := .reference }} {{- $suppressAuthor := .suppressAuthor }} {{- $format := .format }} {{- $pages := .pages }} {{- $partialPath := .partialPath }} {{- $currentPages := "" }} {{- if lt $keyIndex (len $pages) }} {{- $currentPages = index $pages $keyIndex }} {{- end }} {{- /* Add to the collection of cited references */}} {{- $.Page.Scratch.SetInMap "citedBib" $key $reference -}} Citation: {{- if not $suppressAuthor -}} {{- $displayAuthors := $reference.author -}} {{- if not $reference.author }} {{- $displayAuthors = $reference.editor -}} {{- end }} {{- if not $displayAuthors }} {{- i18n "apa_no_author_abbr" | default "n.a." | upper -}} {{- else }} {{- $authorLen := len $displayAuthors }} {{- range $authorIndex, $author := first 2 $displayAuthors }} {{- with $author.given }} {{- end }} {{- with $author.family }} {{ . | markdownify }} {{- end }} {{- if and (eq $authorIndex 0) (gt $authorLen 2) -}} , {{- end }} {{- if and (eq $authorLen 2) (eq $authorIndex 0) -}} & {{- end }} {{- end }} {{- if gt $authorLen 2 }} & al. {{- end }} {{- end }} {{- if eq $format "p" }}, {{- else }} {{- end }} {{- end }} {{- /* Year */}} {{- if and (isset $reference "issued") (isset $reference.issued "date-parts") -}} {{- $dateParts := index $reference.issued "date-parts" }} {{- if gt (len $dateParts) 0 }} {{- $year := index (index $dateParts 0) 0 }} {{- $year -}} {{- end }} {{- end }} {{- with $currentPages -}} {{- if eq $format "p" }} , {{- $formattedPages := (printf "p. %s" .) | safeHTML }} {{- if (findRE "-" .) -}}{{/* If `$pages` contains a dash */}} {{- $formattedPages = (printf "pp. %s" .) | safeHTML }} {{- end }} {{- $formattedPages -}} {{- else }} : {{- . -}} {{- end }} {{- end }} {{- /* Eliminate space between css-hidden citation hover block */ -}} {{ partial $partialPath $reference }}
